НАЗВАНИЕ
========

Описание
--------
Плагин для Google Chrome, добавляющий статистику, генерируемую на основе данных друзей, на страницу профиля вконтакте.

-------------

* Статиска возрастов
* Статиска университетов
* Статиска школ
